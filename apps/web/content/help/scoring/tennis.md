Tennis divisions score with real tennis rules: points build to games, games to sets, with deuce and advantage handled for you. Pick a format when you create the division — **Tour** (best of 3, tie-break sets), **Grand slam** (best of 5, 10-point tie-break in the deciding set), **Fast4** (short sets to 4) or **Doubles no-ad** (a 10-point match tie-break replaces the deciding set).

## Point-by-point

One tap per point. The pad speaks the score the way an umpire would — 15, 30, 40, deuce, advantage — and shows a dot next to whoever is serving. Tie-breaks switch to plain numbers and rotate the serve automatically (one point, then two each).

Mistapped? **Undo last** removes the point and the game state steps back exactly — even across a banked set or match point.

## Set totals

No time to score every point? Switch to **Set totals** and enter each finished set's games. A 7–6 set needs its tie-break points too (that's the number in brackets on the scoreboard — `7–6(5)`). For a deciding match tie-break, enter the tie-break points themselves as the set score.

Both modes feed the same standings — sets won, then set ratio, then game ratio break ties.

## Common questions

**Can I mix modes?** Yes — rally one set point-by-point and enter the next as a total. You just can't enter a total for a set that's mid-rally.

**Who serves first?** The pad starts with the home side. The serve dot is a scoring aid, not a record — it doesn't affect results.
