# Seazn Club brand assets

The mark: **Anton** wordmark in night ink, lime pitch line, red ball resting
at the line's right end. Colours: night `#150b36`, cream `#f5f0e8`, lime
`#a3e635`, ball `#ef4444`.

## Print (shirts, banners, merch)

| File | Use |
| --- | --- |
| `seazn-logo-print.svg` / `-4k.png` | Light backgrounds (white/cream shirts) |
| `seazn-logo-print-cream.svg` / `-4k.png` | Dark backgrounds (night/black shirts) |
| `seazn-logo-print-mono.svg` / `-4k.png` | Single-ink: embroidery, stamps, engraving |

The 4K PNGs are transparent, 4096px wide — ~35 cm at 300 dpi, enough for a
full chest print. The SVGs are the editable masters; the wordmark uses Anton
(Google Fonts, OFL) — if a print shop asks for outlined text, open the SVG in
Illustrator/Inkscape with Anton installed and convert text to outlines, or
hand them the 4K PNG.
